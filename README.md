## Add-ons

### OpenVPN Server
Full-featured OpenVPN server with EasyRSA PKI for secure remote access to Home Assistant and your local network.

**Features:**
- Complete PKI (Public Key Infrastructure) management
- Easy client certificate generation
- Certificate revocation support (CRL)
- Automatic .ovpn file generation
- AES-256-GCM encryption
- Connection monitoring

# OpenVPN Server Add-on Repository

Home Assistant add-on repository containing OpenVPN Server.

## Installation

Add this repository to your Home Assistant instance:

[![Add Repository](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Flaurentboutet%2Fha-openvpn-server)

Or manually:
1. Navigate to **Settings** → **Add-ons** → **Add-on Store**
2. Click **⋮** (menu) → **Repositories**
3. Add: `https://github.com/laurentboutet/ha-openvpn-server`
4. Click **Add**

## Add-ons in this repository

### OpenVPN Server

Full-featured OpenVPN server with PKI management for secure remote access.
